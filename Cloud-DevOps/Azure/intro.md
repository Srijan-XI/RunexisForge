# Azure Introduction

Microsoft Azure is a cloud platform offering compute, storage, databases, AI, and DevOps services across global regions.

## Core Services
- Compute: VMs, App Service, AKS, Functions
- Storage: Blob, Files, Queues, Tables
- Data: SQL Database, Cosmos DB, PostgreSQL/MySQL Flexible Server
- Networking: VNet, Load Balancer, Application Gateway, Front Door
- Identity: Azure AD (Entra ID), Managed Identities
