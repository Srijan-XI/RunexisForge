# Vite User Guide

## Create a project

```bash
npm create vite@latest
cd your-project
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Environment variables

- `.env` files are supported.
- Only variables prefixed with `VITE_` are exposed to the client by default.

## Config

See `Vite/examples/vite.config.ts` for a minimal config example.
