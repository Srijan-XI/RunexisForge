# MX Linux User Guide

## Installation

1. Download from [mxlinux.org](https://mxlinux.org/download-links/)
2. Create bootable USB
3. Boot from USB
4. Run installer
5. Follow wizard

```bash

# Install

# Or use MX Package Installer (GUI)
```
