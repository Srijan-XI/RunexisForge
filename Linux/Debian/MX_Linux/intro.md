# Introduction to MX Linux

## What is MX Linux?

**MX Linux** is a cooperative venture between the antiX and former MEPIS Linux communities. It's a midweight distribution that combines elegant and efficient desktop with simple configuration, high stability, solid performance, and medium-sized footprint.
