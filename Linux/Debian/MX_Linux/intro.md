# MX Linux Introduction

MX Linux is a Debian-based distribution known for its stability, lightweight performance, and MX Tools suite.

## Highlights
- Based on Debian Stable
- Xfce/KDE/Fluxbox editions
- MX Tools for snapshots, boot repair, and drivers
