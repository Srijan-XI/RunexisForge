# Linux Mint User Guide

## Update & Upgrade
- Use Update Manager GUI or terminal:
```
sudo apt update
sudo apt upgrade
```

## Install Software
- Software Manager GUI or:
```
sudo apt install <pkg>
```

## Install Linux Mint
- Download ISO: https://linuxmint.com/download.php
- Create bootable USB, boot, run installer
- Beginner-friendly; choose desktop (Cinnamon/MATE/Xfce)
- More: [../INSTALL.md](../INSTALL.md)

## Snap/Flatpak
- Mint favors Flatpak; enable and install via GUI or `flatpak`.
