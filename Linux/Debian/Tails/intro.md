# Tails Introduction

Tails (The Amnesic Incognito Live System) is a privacy-focused live Linux distribution that routes traffic through Tor and leaves no trace on the host machine by default.

## Highlights
- Based on Debian
- Forces all traffic through Tor
- Amnesic: does not write to disk unless using persistent storage
- Includes privacy tools (Tor Browser, KeePassXC, OnionShare)
