# Introduction to Tails

## What is Tails?

**Tails** (The Amnesic Incognito Live System) is a security-focused Debian-based Linux distribution aimed at preserving privacy and anonymity. It connects to the Internet exclusively through the Tor network and leaves no trace on the computer being used.

## Key Features

### Privacy & Anonymity
- **Tor Network**: All traffic anonymized
- **No Traces**: Amnesia feature clears memory
- **Encrypted Storage**: Persistent volume encryption

### Security Tools
- **Tor Browser**: Anonymous web browsing
- **Encrypted Messaging**: Pidgin with OTR
- **Email Encryption**: Thunderbird with OpenPGP
- **KeePassXC**: Password manager
