# Introduction to Zorin OS

## What is Zorin OS?

**Zorin OS** is a Ubuntu-based Linux distribution designed specifically to make the transition from Windows and macOS to Linux as smooth as possible. It features a familiar desktop environment and comes with pre-installed software for everyday computing tasks.

## Editions

### Zorin OS Core (Free)
- Modern desktop experience
- Pre-installed essential apps
- Based on Ubuntu LTS

### Zorin OS Lite (Free)
- For older/slower hardware
- XFCE desktop environment
- Minimal resource usage

### Zorin OS Pro ($39)
- Premium desktop layouts
- Advanced productivity apps
- Professional creative software
- Priority support

### Zorin OS Education
- For schools and students
- Educational software included
- Parental controls

---

## Key Features

- **Zorin Appearance**: Desktop layout customization
- **Windows App Support**: Wine and PlayOnLinux pre-installed
- **Software Center**: Easy app installation
- **Touch Gestures**: Multi-touch support
- **Built-in Apps**: LibreOffice, GIMP, Firefox, Thunderbird

---

## System Requirements

### Minimum
- **CPU**: 1 GHz Dual Core
- **RAM**: 2 GB (Core), 1 GB (Lite)
- **Storage**: 10 GB

### Recommended
- **CPU**: 2 GHz Dual Core or better
- **RAM**: 4 GB+
- **Storage**: 20 GB+ SSD
