# Zorin OS Introduction

Zorin OS is an Ubuntu-based distribution focused on ease of use and a Windows-like desktop experience.

## Highlights
- Based on Ubuntu LTS for stability
- Zorin Appearance for desktop layouts (Windows/macOS-like)
- Flatpak support and Zorin Connect for device integration
