# Parrot OS User Guide

## Update & Upgrade
```
sudo apt update
sudo apt upgrade
```

## Install Tools
```
sudo apt install parrot-tools
```

## Install Parrot OS
- Download ISO: https://www.parrotsec.org/download/
- Create bootable USB, boot installer
- Choose Home, Pentesting, or Security editions
- More: [../INSTALL.md](../INSTALL.md)

## Privacy
- Includes privacy tools; review settings and browser profiles.
