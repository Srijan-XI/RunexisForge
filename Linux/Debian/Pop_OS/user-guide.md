# Pop!_OS User Guide

## Installation

1. Download ISO from [pop.system76.com](https://pop.system76.com/)
2. Choose Intel/AMD or NVIDIA version
3. Create bootable USB
4. Boot and install
5. Restart

---

## Pop Shell (Tiling)

### Enable Auto-Tiling

```bash
Super + Y (toggle tiling)
```bash

### Keyboard Shortcuts

```bash
Super + Enter - Terminal
Super + / - Launcher
Super + O - Workspaces
Super + Arrow Keys - Move focus
Super + Shift + Arrow Keys - Move windows
```bash

---

## Package Management

```bash
# Update
sudo apt update && sudo apt upgrade

# Install
sudo apt install <package>

# Flatpak (pre-installed)
flatpak install <app>
```bash

---

## Gaming

### Install Steam

```bash
# Via Pop!_Shop or
sudo apt install steam
```bash

### GameMode (pre-installed)

```bash
# Runs automatically with games
gamemoderun <game>
```bash
