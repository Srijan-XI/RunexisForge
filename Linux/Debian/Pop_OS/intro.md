# Introduction to Pop!_OS

## What is Pop!_OS?

**Pop!_OS** is a free, open-source Linux distribution developed by System76, designed for developers, makers, and computer science professionals. Based on Ubuntu, it offers a streamlined experience with a focus on productivity and creativity.

## Key Features

### Pop Shell
- **Auto-tiling**: Automatic window tiling
- **Keyboard shortcuts**: Efficient workflow
- **Workspace management**: Easy multi-desktop switching

### Built-in Tools
- **Pop!_Shop**: Curated application store
- **NVIDIA ISO**: Dedicated ISO with NVIDIA drivers
- **Power Profiles**: Easy battery/performance management
- **Firmware Manager**: Easy hardware updates

---

## Editions

### Pop!_OS (Intel/AMD)
- For Intel or AMD graphics
- Standard kernel

### Pop!_OS (NVIDIA)
- Pre-installed NVIDIA drivers
- Optimized for NVIDIA GPUs

---

## System Requirements

- **CPU**: 64-bit x86 processor
- **RAM**: 4 GB minimum (8 GB+ recommended)
- **Storage**: 20 GB minimum
- **GPU**: Intel, AMD, or NVIDIA
