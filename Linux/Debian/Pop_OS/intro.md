# Pop!_OS Introduction

Pop!_OS is an Ubuntu-based distribution by System76 tailored for developers, with tiling window management and hardware acceleration focus.

## Highlights
- Auto-tiling window manager and keyboard-driven workflow
- NVIDIA and AMD images for correct drivers
- COSMIC desktop environment with GNOME base
