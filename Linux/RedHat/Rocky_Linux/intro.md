# Rocky Linux Introduction

Rocky Linux is a community-driven, RHEL-compatible distribution providing a stable, enterprise-ready platform.

## Highlights
- Bug-for-bug compatible with Red Hat Enterprise Linux
- 10-year support lifecycle per major release
- Uses `dnf` for package management
- Strong focus on stability and predictability
