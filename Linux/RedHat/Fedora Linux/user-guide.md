# Fedora Linux User Guide

## Update & Upgrade

```bash
sudo dnf upgrade --refresh
```bash

## Install Software

```bash
sudo dnf install <pkg>
```bash

## Install Fedora Linux

- Download ISO: <https://getfedora.org/>
- Create bootable USB, boot installer
- Cutting-edge packages; new Fedora every ~6 months
- More: [../INSTALL.md](../INSTALL.md)

## Toolbox (Developers)

- Use `toolbox` for containerized dev environments.
