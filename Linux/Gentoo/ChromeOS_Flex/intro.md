# ChromeOS Flex Introduction

ChromeOS Flex is Google's Chromium OS-based system for repurposing existing PCs and Macs with a lightweight, cloud-managed OS.

## Highlights
- Fast boot and simple management
- Integrates with Google Workspace for enterprise/education
- Based on Chromium OS with verified boot
