# ChromeOS Flex User Guide

## Install
- Create a bootable USB using Chromebook Recovery Utility
- Select ChromeOS Flex in the utility
- Boot target device from USB; try live mode or install

## Login and Setup
- Sign in with Google account
- For enterprise, enroll with Google Admin console

## Apps
- Web apps and PWAs via Chrome
- Limited Android app support (not included)

## Updates
- Automatic background updates; minimal user action

## Management
- Use Google Admin console for policies, Wi-Fi, proxies, and kiosk mode

## Troubleshooting
- Powerwash (factory reset) from settings if issues occur
- Recreate USB if installation fails
