# BlackArch User Guide

## Setup Repos

- Follow official instructions to enable BlackArch repositories or install the ISO.

## Update & Upgrade

```bash
sudo pacman -Syu
```bash

## Search & Install Tools

```bash
pacman -Ss blackarch
sudo pacman -S <tool>
```bash

## Install BlackArch

- Install Arch first: <https://archlinux.org/download/>
- Enable BlackArch repos (see official guide at blackarch.org)
- More: [../INSTALL.md](../INSTALL.md)

## Notes

- Use responsibly; many tools require elevated permissions.
