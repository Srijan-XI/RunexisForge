# Garuda Linux User Guide

## Update & Upgrade
```
sudo pacman -Syu
```

## Install Software
```
sudo pacman -S <pkg>
```

## Snapshots (Btrfs)
- Manage snapshots via GUI tools; rollback via boot menu.

## Install Garuda Linux
- Download ISO: https://garudalinux.org/downloads.html
- Create bootable USB, boot installer
- Includes Btrfs with snapshots and GUI tools
- More: [../INSTALL.md](../INSTALL.md)

## AUR
- Use an AUR helper (e.g., `yay`) for community packages.
