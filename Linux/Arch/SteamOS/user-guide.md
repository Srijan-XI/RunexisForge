# SteamOS User Guide

## Updates
- In Game Mode: Settings > System > Check for Updates
- Choose Stable/Beta channels for system and Steam client

## Desktop Mode
- Switch via Power menu > Desktop Mode
- Use Discover for packages or `pacman` for SteamOS 3 (Arch-based on Deck)

## Game Compatibility
- Proton versions managed per game; set in game Properties
- Check ProtonDB for community reports

## External Displays and Peripherals
- Use USB-C hub/dock for HDMI/DP and USB
- Configure resolutions in Display settings

## Storage
- microSD support; format to ext4 in SteamOS
- Move games between internal and SD via Storage settings

## Troubleshooting
- Clear shader cache if a game misbehaves
- Re-image with SteamOS recovery if system is corrupted (see Valve guide)
