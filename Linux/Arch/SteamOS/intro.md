# SteamOS Introduction

SteamOS (used on Valve's Steam Deck) is a Debian-based distribution tailored for gaming with the Steam client and Proton compatibility layer.

## Highlights
- Gaming-focused, ships with KDE Plasma on desktop mode
- Proton for running many Windows games
- Game Mode UI optimized for controllers
