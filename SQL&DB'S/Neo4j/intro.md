# Neo4j Introduction

Neo4j is a native graph database built to store and query highly connected data using the property graph model and Cypher query language.

## Key Features
- Nodes and relationships with properties
- ACID transactions and native graph storage
- Powerful pattern matching via Cypher
- Drivers for many languages and Graph Data Science library
