# Remix Introduction

## Overview
Remix is a full-stack React framework focused on web standards, leveraging server rendering, nested routes, and progressive enhancement.

## Key Features
- Nested routing with loaders and actions for data
- Built-in forms with progressive enhancement
- Server-first by default; adapts to edge runtimes
- TypeScript support and file-based routing
- Works with various deployments (Node, Vercel, Cloudflare, Fly.io)

## Common Use Cases
- Data-driven apps with strong form handling
- SEO-friendly sites with fast streaming responses
- Edge-rendered apps for low latency
