# Unreal Engine — User Guide

## 1) Install (Windows)
- Install Epic Games Launcher
- Install Unreal Engine (UE5 recommended)

## 2) Create a Project
1. Open Unreal Engine → **Games** template
2. Choose Blueprint or C++ project
3. Pick a starter template (First Person / Third Person)

## 3) First Steps
- Create an Actor Blueprint and place it in the level
- Use the **Details** panel to tweak properties
- Press **Play** to test
