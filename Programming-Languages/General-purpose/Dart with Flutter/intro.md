# Dart with Flutter Introduction

## Overview

Dart is a client-optimized language, and Flutter is its UI toolkit for building natively compiled apps for mobile, web, and desktop from a single codebase.

## Key Features

- Hot reload for rapid UI iteration
- Declarative widget tree
- Single codebase targeting iOS, Android, web, and desktop
- Strong typing with sound null safety
- Rich ecosystem of packages via pub.dev

## Common Use Cases

- Mobile apps with polished UI
- Progressive web apps
- Desktop apps for Windows, macOS, Linux

## Essentials

- Projects managed with `flutter` CLI and `pubspec.yaml`
- Widgets compose UI; state managed with setState or state management libs (Provider, Riverpod, BLoC)
- Uses Dart's async/await and Streams for async work
