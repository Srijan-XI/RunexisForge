# Ruby — Introduction

## What is Ruby?

Ruby is a dynamic, object-oriented programming language known for readable syntax and developer happiness. It’s widely used for scripting, automation, and web development (especially with Ruby on Rails).

## Why Ruby?

- Very readable syntax
- Great for scripting and quick prototyping
- Huge ecosystem via RubyGems

## Where Ruby is used

- Web apps (Ruby on Rails, Sinatra, Cuba)
- Automation scripts
- Dev tooling

## Learning Path

1. Install Ruby and run a hello-world script.
2. Learn variables, control flow, arrays/hashes.
3. Learn methods, classes, modules.
4. Use Bundler and gems.
