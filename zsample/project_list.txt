Project List

Project List divided by code like for beginner level project are noticed by 'B',
Intermediate level project are noticed by 'I', Advanced level project are noticed by 'A' and All level project are noticed by 'AL'.

For example: TO-DO 'B' means this is a beginner level project, library management system 'I' means this is an intermediate level project,
Chat application 'A' means this is an advanced level project, AI based code generator and 'AL' means this project is for all levels.
