# Godot — User Guide

## 1) Install (Windows)
Download Godot from the official website and run the editor.

## 2) Create a Project
1. Open Godot → New Project
2. Choose a folder and renderer
3. Create and edit

## 3) Your First Script
Attach a script to a Node (e.g., Node2D) and run the scene.

Example: `Godot/examples/hello.gd`

## Examples & Practice
- Examples: `Godot/examples/`
- Practice: `Godot/questions/`
