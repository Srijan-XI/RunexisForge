# Q1: WSL Setup Checklist

Goal: set up WSL and confirm it works.

1. Install WSL and Ubuntu.
2. Run `wsl --list --verbose` and confirm the distro is running.
3. In Ubuntu, run:
   - `sudo apt update`
   - `uname -a`
4. Create `~/projects/hello` and add a file `hello.txt`.
5. From inside WSL, open the folder in Explorer with `explorer.exe .`.

Bonus:
- Install `git` and run `git --version`.
