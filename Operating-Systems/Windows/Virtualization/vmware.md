VMware products vary by platform and license. Common desktop options are:

- **VMware Workstation Pro** (Windows/Linux)
- **VMware Fusion** (macOS)

## Create your first VM (typical flow)

1. **Create a New Virtual Machine**
2. Select an installer ISO
3. Pick OS type/version
4. Allocate CPU/RAM and disk
5. Start the VM and complete installation

## Snapshots and clones

- **Snapshots**: safe restore points
- **Clones**: copy a VM for experiments

## Networking basics

- NAT / Bridged / Host-only (same concepts as VirtualBox)

## Automation

- VMware ecosystems often use **PowerCLI** (especially in enterprise environments)
