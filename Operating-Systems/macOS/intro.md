# macOS Introduction

## Overview
macOS is Apple's desktop operating system, known for its UNIX foundation, hardware integration, and productivity ecosystem.

## Key Features
- UNIX-based with BSD userland and launchd
- Tight integration with Apple hardware (Apple Silicon, T2)
- Built-in security: Gatekeeper, SIP, FileVault
- Development ready: comes with Terminal, zsh, and supports Homebrew
