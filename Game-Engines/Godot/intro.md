# Godot — Introduction

## Overview

Godot is a free and open-source game engine for 2D and 3D games. It’s known for a lightweight editor, fast iteration, and a clean scene/node system. The most common scripting language is **GDScript** (Python-like), and it also supports C#.

## Why Godot?

- Free and open-source
- Excellent 2D workflow
- Simple scene/node architecture
- Great for learning game dev fundamentals

## Core Concepts

- **Scene**: Reusable collection of nodes
- **Node**: Building block (sprites, physics bodies, scripts, UI)
- **Signals**: Event system for communication
- **Script**: Behavior attached to nodes
