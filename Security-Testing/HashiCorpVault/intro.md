# HashiCorp Vault Introduction

HashiCorp Vault is a secrets management and encryption platform for securely storing, accessing, and distributing secrets and sensitive data.

## Key Features
- Dynamic secrets for databases and cloud providers
- Encryption as a service (transit)
- PKI engine for certificates
- Access via policies tied to auth methods (tokens, AppRole, Kubernetes, OIDC)
